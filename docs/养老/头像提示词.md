# 公众号头像 AI 生图提示词（豆包专用）

> 账号定位：社保养老政策 + 中老年健康养生
> 受众：50+ 中老年群体
> 调性：亲切、可信、温暖、官方感
> 生图工具：豆包（PC 端 / App 端 → 图像生成）
> 通用要求：圆形构图友好、小尺寸辨识度高、不含真人脸、不含文字、不含商标

---

## 使用方法（豆包专属）

1. 打开豆包 → 输入框旁切换到「图像生成」或直接说"帮我生成一张图"
2. 复制下面任一 Prompt 整段粘贴
3. **比例选 1:1**（豆包生图设置里可选）
4. **风格**：选「插画」或「不指定」（让 prompt 内的风格关键词主导）
5. 一次生 4 张，挑居中、干净、小尺寸下识别度最高的那张
6. 不满意就在 prompt 后追加："再温暖一点 / 颜色再浅一点 / 元素再少一点 / 居中一点"——豆包支持自然语言追问改图

## 风格一：暖阳小屋风（强烈推荐，亲和力最强）

```
生成一张微信公众号头像，1:1 正方形构图，圆形居中布局，扁平矢量插画风格。
画面内容：一座圆顶橙色小屋静静坐落在画面正中下方，屋顶上方有一轮饱满温暖的橙黄色太阳正在升起，太阳周围散发出柔和的米黄色光晕，光线柔软不刺眼。小屋下方用一道淡淡的心形弧线轻轻托起，象征"家与守护"。
色彩：主色调暖橙、奶油黄、淡米色，背景是干净的奶白色，整体色温偏暖。
风格要求：扁平插画、线条圆润、画面治愈、无任何文字、无水印、无logo、无真人、无英文字母。画面构图居中，四周留白舒适，小尺寸缩略图下依然清晰可辨。
```

```

画面主体：画面正中是一枚朱砂红色的圆形中式印章，印章中央雕刻一个手写书法体的"安"字（笔触圆润饱满，带毛笔书法的留白与飞白质感）。印章边缘点缀极简的祥云纹和如意纹。
```

## 豆包改图微调话术（生完图觉得差点意思时直接说）

- "颜色再暖一点，加点黄"
- "元素太多了，简化一下，只保留小屋和太阳"
- "背景太花了，换成纯净奶白色"
- "线条再粗一点 / 再细一点"
- "整体往中间收一点，留白多一点"
- "把红色再调正一点，要朱砂红不要番茄红"
- "去掉画面里的文字 / 字母 / 符号"
- "光晕弱一点，不要太亮"

---

## 避坑清单

1. **不要在 prompt 里要求出文字**——豆包写中文字常出错，头像本来也不该带字。
2. **不要参考剧照/明星/影视/综艺截图/品牌商品图**——侵权风险，平台审核也过不了。
3. **生成后裁切成正方形再上传**，公众号头像会自动圆形裁切，所以重要元素一定要居中、四角可裁掉。
4. **如果豆包多次生成都带了字母或水印**，prompt 末尾加一句："严格禁止画面出现任何文字、字母、数字、符号、水印"。
5. **头像与公众号名色调统一**：选定头像后，把主色（如暖橙 / 朱砂红 / 橄榄绿）记下来，后面做封面图统一沿用，整个号视觉就立住了。

## 推荐发布流程

1. 三个风格各生 4 张 → 共 12 张候选
2. 缩到 100×100 像素预览（模拟手机微信里看到的大小）
3. 还能一眼认出主体的，就是合格头像
4. 选 1 张主用，1 张备用（半年换一次保持新鲜感）
